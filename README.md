# INO_SV660N

**Описание**

Этот репозиторий содержит примерные конфигурационные файлы и вспомогательную документацию для связки
LinuxCNC + EtherCAT + Inovance SV660N. Цель — собрать удобную отправную точку для настройки управления приводами Inovance
SV660N через EtherCAT в среде LinuxCNC.

**Содержание репозитория**
- `cfg/` : шаблоны `*.ini` для LinuxCNC (обычно содержит настройки осей, координатные системы и др.).
- `hal/` : примерные `*.hal`-файлы для связывания LinuxCNC с EtherCAT и драйверами.
- `scripts/` : вспомогательные скрипты для инициализации, тестов и утилит.
- `docs/` : заметки по настройке, ссылки на мануалы и рекомендации.

Примечание: в текущем минимальном репозитории может не быть всех перечисленных папок — они указаны как рекомендуемая структура.

**Требования**

- **ОС**: Linux с real-time ядром (рекомендовано `rt-preempt`).
- **LinuxCNC**: версия, совместимая с вашей системой (установите через пакетный менеджер или из исходников).
- **EtherCAT Master**: один из вариантов — IgH EtherCAT master или решение на базе SOEM; нужна совместимая конфигурация мастера.
- **Приводы**: Inovance SV660N — убедитесь, что у вас есть доступ к руководству пользователя и параметрам привода.

**Установка и базовая настройка**

1. Установите LinuxCNC и real-time ядро согласно вашей дистрибутивной документации.
2. Установите и настройте EtherCAT master (IgH или SOEM). Проверьте обнаружение узлов EtherCAT и их состояния.
3. Скопируйте шаблон `*.ini` в свою рабочую директорию и адаптируйте параметры осей, скорости и координат.
4. Настройте `*.hal`-файлы: подключите сигналы LinuxCNC к EtherCAT PDO (входы/выходы привода), добавьте интерфейс аварийной остановки и индикаторы состояния.
5. Запустите LinuxCNC: `linuxcnc /path/to/your.ini` и убедитесь, что связи с приводами установлены и статусы корректны.

Пример команды для запуска (локально):

```
linuxcnc /path/to/your.ini
```

**Особенности работы с Inovance SV660N через EtherCAT**

- Уточняйте соответствие PDO/SDO в документации Inovance и вашего EtherCAT-мастера.
- Перед прямым управлением убедитесь, что параметры привода (например, режим управления, масштаб энкодера, направление вращения) заданы корректно.
- Рекомендуется тестировать сначала в медленном режиме и с ограничением тока/скорости.
